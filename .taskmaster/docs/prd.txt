# Fajrak Flutter — UI Redesign PRD

## Project Overview
Complete UI redesign of the Fajrak personal finance app to match the fajrak-design system (Stripe visual language + Apple HIG + Material 3). The app is a bilingual (AR/EN) RTL-first Flutter app for tracking income, expenses, budgets, debts, goals, and investments.

## Design System
- Colors: primary #1A5F8A, accent #D4A843, income #2A7D4F, expense #B83232
- Font: IBM Plex Sans Arabic (GoogleFonts)
- Elevation: 0 always — cards use 1px border, no shadows
- FAB: gold (#D4A843) — only shadow in the app
- All padding: EdgeInsetsDirectional (start/end), never left/right
- Loading: shimmer always, never CircularProgressIndicator for data
- Animations: Emil Kowalski curves — Cubic(0.23,1,0.32,1) easeOut
